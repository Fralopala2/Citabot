Política de Privacidad de Citabot

Última actualización: 16 de septiembre de 2025

Citabot, desarrollada por PACOALDEV, se compromete a proteger la privacidad de sus usuarios. Esta aplicación permite consultar fechas y horas disponibles para gestionar citas previas de la ITV en España.

1. Recogida y uso de datos

  • Datos personales: Citabot no solicita ni almacena datos personales identificativos como nombre, dirección o DNI.
  • Datos técnicos: Se recopila información técnica estándar (tipo de dispositivo, sistema operativo, etc.) para mejorar el funcionamiento de la app.
  • Notificaciones: Utilizamos Firebase Cloud Messaging para enviar notificaciones push. El identificador de dispositivo (token FCM) se almacena únicamente para este fin.(EN DESARROLLO)
  • Publicidad: La app muestra anuncios de Google AdMob, que puede recopilar datos anónimos sobre el uso y mostrar publicidad personalizada o no personalizada según la configuración del usuario.

2. Servicios de terceros

  • Google AdMob: Para mostrar anuncios y monetizar la app.
  • Firebase (Google): Para notificaciones push y análisis de uso.(EN DESARROLLO)

3. Derechos del usuario

  • Puedes desinstalar la app en cualquier momento.
  • Puedes desactivar la personalización de anuncios desde los ajustes de Google en tu dispositivo.
  • Si deseas ejercer tus derechos de acceso, rectificación o supresión de datos relacionados con notificaciones, contacta con nosotros.
